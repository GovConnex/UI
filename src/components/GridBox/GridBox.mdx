# GridBox
[Source code](https://github.com/GovConnex/UI/tree/main/src/components/GridBox)

## Overview

`GridBox` is a higher-order component that extends the functionality of the `Box` component. It provides a responsive grid layout with configurable column counts and gaps for each breakpoint.

By default, the grid will have 4 columns for the "sm" breakpoint, 8 columns for the "md" breakpoint, and 12 columns for the "lg" breakpoint. There will be column gaps of 8px for the "sm" breakpoint, 16px for the "md" breakpoint, and 24px for the "lg" breakpoint. There will be no row gaps.

## Usage

Here is a simple example of how to use `GridBox`:

```jsx
<GridBox mt="spacing.md">
  // Child components go here
</GridBox>
```
