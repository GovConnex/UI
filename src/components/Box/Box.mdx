## Overview
`Box` manages layout of its children given a custom style 

## Usage

Here is a simple example of how to use `Box`
```jsx
<Box cs={{padding:"20px", display:"flex"}}>
  Your content
</Box>
```

### Responsive values

With `cs` you can also access theme and breakpoints!
`sm`, `md`, `lg`, `xl`

#### Order matters*

```jsx
<Box cs={{ backgroundColor:"primary.brand.900", md:{padding:"spacing.sm"}, lg:{padding:"spacing.lg"} }} >
  Your content
</Box>
```
