## Component API

<Props />

Additional props passed into `Tabs.Tab` will be forwarded along to the underlying
button element.



```jsx
<Tabs value="defaultTab">
  <Tabs.Tab onClick={() => {doSomthing()}} key="3" value="defaultTab" label="Tab1"/>
</Tabs>
```
